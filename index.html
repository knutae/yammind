The object of the game is to find the hidden color code.
  • Click on each hole and select a color from the popup menu.
  • Click on "Check" to test your current guess.
  • Black and white indicators to the right will show how close your guess was:
    • indicates that a correct color is in the correct position.
    • indicates that a correct color is in the wrong position.
  • Use this feedback to guess the secret code in as few attempts as possible!
